## Development
### How did I start the project
```sh
pip install mkdocs
mkdocs --version
# mkdocs, version 1.1.2 from /path/to/python3.8/site-packages/mkdocs (Python3.8)
mkdocs new mkdocs_practice
```

### Dynamic Serving
```sh
mkdocs serve
```

### Deploy as github pages
```sh
mkdocs gh-deploy
```
